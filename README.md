# Footwear Shop

Footwear Shop este o aplicatie de gestionare a produselor dintr-un magazin de incaltaminte, cu optiuni de afisare a produselor in timp real, salvarea si incarcarea datelor intr-un fisier text si a posibilitatii de adaugare stoc sau vinderea unui numar ales de bucati dintr-o gama aleasa.

## Tehnologii si concepte utilizate

- Clase abstracte, polimorfism, interfete;
- Colectii, expresii lambda;
- Tratarea exceptiilor si crearea unor exceptii personalizate;
- Interfata grafica, folosind Windows Forms si .NET Framework

## Programe si limbaje utilizate

- Visual Studio Community 2022 (folosind limbajul C#)
- .NET Framework 4.7.2

## Cum sa rulati programul?

1. (Nu necesita programul Visual Studio Community 2022!!!) 
Faceti dublu-click pe executabilul "Footwear_Shop.exe" din locatia "Footwear_Shop\bin\Debug";

2. (Necesita programul Visual Studio Community 2022 sau un alt program compatibil!!!) 
Faceti dublu-click pe fisierul "Footwear_Shop.sln", apoi apasati tasta F5 ca sa rulati codul si sa va porneasca aplicatia.

## Ce am invatat din realizarea acestui proiect?

- Modul de gestionare si manipulare a datelor dintr-un fisier text;
- Cum sa creezi o aplicatie Windows Forms, folosind framework-ul .NET;
- Personalizarea elementelor vizuale si interactive dintr-o interfata grafica;
